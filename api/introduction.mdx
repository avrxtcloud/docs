# AVRXT API Endpoints

Base URL:

```bash
https://avrxt.in
```

This reference covers all routes currently implemented under `src/app/api` in `avrxt-in`.

## Implemented Endpoints

- `POST /api/subscribe`
- `POST /api/contact`
- `POST /api/hireme`
- `GET /api/spotify/auth`
- `GET /api/spotify/callback`
- `GET /api/spotify/now-playing`
- `GET /api/health`
- `GET /api/status`

## Method Summary

- `POST`: `subscribe`, `contact`, `hireme`
- `GET`: `spotify/auth`, `spotify/callback`, `spotify/now-playing`, `health`, `status`
- `OPTIONS`: available for `subscribe`, `contact`, `hireme`
